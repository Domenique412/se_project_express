# WTWR (What to Wear?): Back End

### Description

The back-end project is focused on creating a server for the WTWR application.

### Testing

The goal of this project is to create a secure, user-friendly backend for managing clothing items and
personal profiles. Essentially, it helps users keep track of their wardrobe and interact with the system
safely, while protecting sensitive data like passwords and personal information.

### Techs Used

Express, MongoDB, Mongoose, ESLint with Airbnb, Validator, Postman,
