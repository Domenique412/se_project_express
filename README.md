# WTWR (What to Wear?): Back End

### Description

A RESTful API server for the WTWR (What to Wear?) application a weather based clothing recommendation platform. Built with Node.js and Express, this backend handles user authentication, clothing item management, and integrates with a React frontend deployed in production with Nginx and PM2.

## Deployed Application

- **Domain:** https://wtwrdom.mooo.com
- **API:** https://api.wtwrdom.mooo.com

## Repository

- **Frontend:** https://github.com/Domenique412/se_project_react

### Testing

The goal of this project is to create a secure, user friendly backend for managing clothing items and
personal profiles. Essentially, it helps users keep track of their wardrobe and interact with the system
safely, while protecting sensitive data like passwords and personal information.

### Techs Used

- **Server:** Node.js, Express.js
- **Database:** MongoDB, Mongoose
- **Authentication:** JSON Web Tokens (JWT), bcryptjs
- **Validation:** Celebrate, Joi, Validator
- **Logging:** Winston, Express-Winston
- **Environment:** dotenv
- **Process Management:** PM2
- **Web Server:** Nginx with SSL (Let's Encrypt)
- **Development:** ESLint (Airbnb), Nodemon, Postman
